# 🚀 PathFinder – AI Career Guidance Platform

PathFinder is a modern web application designed to help students (Class 9–12) choose the right career path using an interactive quiz and AI-powered guidance.

It provides personalized recommendations based on user interests, strengths, and academic preferences.

---

## ✨ Features

* 🎯 Career Stream Recommendation (Science / Commerce / Arts)
* 🤖 AI Chatbot for career guidance
* 📊 Personalized quiz-based analysis
* 🏫 College & Scholarship information
* 📄 PDF result generation & sharing
* 🔐 User authentication (Supabase)
* 📱 Fully responsive UI

---

## 🛠️ Tech Stack

* React (Vite)
* TypeScript
* Tailwind CSS
* Supabase (Database & Auth)
* Framer Motion (Animations)

---

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone <your-repo-link>
cd pathfinder
```

---

### 2. Install dependencies

```bash
npm install
```

---

### 3. Setup environment variables

Create a `.env` file in the root directory:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_anon_key
AI_API_KEY=your_ai_api_key
```

---

### 4. Run the project

```bash
npm run dev
```
